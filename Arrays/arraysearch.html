JavaScript Array Search

indexOf()

The indexOf() method searches an array for an element value and returns its poition.

let cars = ["Bmw", "Toyota", "Audi", "Honda"];
alert(cars.indexOf("Audi") + 1);
//adding +1 because array starts from the zero.

array.indexOf() returns -1 if the item is not found

If the item is present more than once, it returns the position of the first occurrence.

array.indexOf(item, start)
item Required. The item to search for.
start. Where to start the search. Negative value will start at the given position counting from the end, and search to the end.

lastIndexOf()

lastIndexOf()is the same as array.indexOf() but returns the position of the last occurence of the specified element

array includes()

ECMAScript 2016 introduced array.includes() to arrays. This allows us to check if an element is present in an array (including Nan, unlike indexOf)

const vegetables = ["Potato", "Tomato", "Carrot", "Onion"];
alert(vegetables.includes("tomato"));
Syntax: array.includes(search-item)

.includes() method is case-sensitive.
.includes() is not supported in internet explorer.

Array find()

The method returns the value of the first array element that passes a test function.

const numbers = [4, 9, 16, 25, 29];
alert(numbers.find(number => number > 18));

Note that the function takes 3 arguments
The item value
The item index
The array itself

'undefined' if no elements satisfy the condition.

Array findIndex()

The findIndex method returns the index of the first array element that passes a test function. 

const numbers = [4, 9, 16, 25, 29];
let first = numbers.findIndex(numbers => numbers > 18);
alert(first);

Note that the function takes 3 arguments
The item value
The item index
The array itself

Array findLast()

The findLast() method returns the value of the last array element that passes a test function.

const numbers = [4, 9, 16, 25, 29, 19, 36, 15, 18];
let first = numbers.findLast(numbers => numbers > 18);
alert(first);
findLast() is an ES2023 feature.

Array findLastIndex()

The findLastIndex() method returns the index of the last array element that passes a test function.

const numbers = [4, 9, 16, 25, 29, 19, 36, 15, 18];
let first = numbers.findLastIndex(numbers => numbers > 18);
alert(first);

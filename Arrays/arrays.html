Arrays

An array is a special variable, which can hold more than one value:

1. Creating Arrays:

  • Imagine an array like a shopping list that can hold multiple items.
  • Use square brackets [] to create an array.
  • Inside the brackets, list the items separated by commas:
const colors = ["red", "green", "blue"];
const numbers = [10, 20, 30];

2. Accessing Elements:

  • Each item has a position called its index (starts from 0).
  • Use the index in square brackets to access specific items:

click on specific code in order to work.

const firstColor = ["red", "green", "blue"];
alert(firstColor);
const secondNumber = [10, 20, 30];
alert(secondNumber[0]);

Changing an Array Element

const cars = ["bmw","mercedes", "audi"]
alert(cars);
cars[0]="changan";
cars[1]="toyota";
cars[2]="honda";
alert(cars);

Arrays Mehtods

Array Length:

The .length property tells you how many items are in the array:

const cars = ["bmw","mercedes", "audi"]
alert(cars.length);

Array Push

const colors= ["red", "green", "blue"];
colors.push("yellow");
alert (colors);

Array toString

The JavaScript method toString() converts an array to a string of (comma separated) array values.

const colors= ["red", "green", "blue"];
message = `type of array is change to '${typeof(colors.toString())}' & string value is ${colors.toString()}`;
alert (message);

Array join

The JavaScript method join() joins all array elements into a string.
It behaves just like toString(), but in addition you can specify the separator: 

const cars=["rolls", "lambo", "bugg"];
carsjoin = cars.join(" & ");
alert(carsjoin);

Popping and Pushing

When you work with arrays, it is easy to remove elements and add new elements.
This is what popping and pushing is:
Popping items out of an array, or pushing items into an array.

Array pop()

The pop() method removes the last element from the array.

const cars=["rolls", "lambo", "bugg"];
  cars.pop();
  alert(cars);

Array push()

The push() method add the element at the end.

const cars=["rolls", "lambo", "bugg"];
cars.push("bmw");
alert(cars);

Shifting Elements

Shifting is equivalent to popping, but working on the first element instead of the last.

Not providing the code samples because it is no needed. It is opposite to pop() & push()

Array shift()

The shift() method removes the first array element and "shifts" all other elements to a lower index. The shift() method returns the value that was "shifted out": 

const fruits = ['apple', 'banana', 'orange', 'grape'];
fruits.shift();
alert(fruits);

Array unshift()

The unshift() method adds a new element to an array (at the beginning), and "unshifts" older elements: The unshift() method returns the new array length: 

const fruits = ['apple', 'banana', 'orange', 'grape'];
fruits.unshift("mango");
alert(fruits);

Array delete()

Using delete() leaves undefined holes in the array. Use pop() or shift() instead. 

const fruits = ['apple', 'banana', 'orange', 'grape'];
delete fruits[0];
alert(fruits);

Array concat()

The concat() method creates a new array by merging (concatenating) existing arrays: 

const boysName= ['Ahmad', 'Ali', 'Umer', 'Ali'];
const girlsName = ['Aisha', 'Amina', 'Khadija', 'Fatima'];
alert(boysName.concat(girlsName));

The concat() method does not change the existing arrays. It always returns a new array.
The concat() method can take any number of array arguments.

const colors= ["red", "green", "blue"];

The JavaScript method join() joins all array elements into a string.
It behaves just like toString(), but in addition you can specify the separator: 

const cars=["rolls", "lambo", "bugg"];
carsjoin = cars.join(" & ");

Popping and Pushing

When you work with arrays, it is easy to remove elements and add new elements.
This is what popping and pushing is:
Popping items out of an array, or pushing items into an array.

Array pop()

The pop() method removes the last element from the array.

const cars=["rolls", "lambo", "bugg"];
  cars.pop();
  alert(cars);

Array push()

The push() method add the element at the end.

const cars=["rolls", "lambo", "bugg"];
cars.push("bmw");
alert(cars);

Array shift()

The shift() method removes the first array element and "shifts" all other elements to a lower index. The shift() method returns the value that was "shifted out": 

const fruits = ['apple', 'banana', 'orange', 'grape'];
fruits.shift();
alert(fruits);

Array unshift()

The unshift() method adds a new element to an array (at the beginning), and "unshifts" older elements: The unshift() method returns the new array length: 

const fruits = ['apple', 'banana', 'orange', 'grape'];
fruits.unshift("mango");
alert(fruits);

Array delete()

Using delete() leaves undefined holes in the array. Use pop() or shift() instead. 

const fruits = ['apple', 'banana', 'orange', 'grape'];
delete fruits[0];
alert(fruits);

Array concat()

The concat() method creates a new array by merging (concatenating) existing arrays:

const teachersName = ['Maaz', 'Zarina', 'Amna', 'Hira'];
alert("All Names: " + boysName.concat(girlsName, teachersName));

The concat() method does not change the existing arrays. It always returns a new array.

Array copyWithin()

The copyWithin method in JavaScript copies part of an array to another location within the same array without changing its length. It takes three arguments: `target`, `start`, and `end`.

'target': Index at which to copy the sequence to.
'start': Index at which to start copying elements from. 'end'(optional): Index at which to end copying elements from. 

const fruits = ['apple', 'banana', 'orange', 'grape', 'mango', 'cherry','pineapple'];
alert(fruits.copyWithin(2,0));

The copyWithin method overwrites the existing values.

The copyWithin method does not add items to the array.

The copyWithin method does not change the length of an array.

If there is no end specified, so it copies up to the end of the array.

Array flat()

The flat() method in Javascript is used to "flatten" an array, meaning it reduces the nesting of arrays. It returns a new array with all sub-array elements concatenated into it recursively, up to specified depth.

const cars =[
              "Ford", "Porsche",
              ["BMW", ['series 1', 'X7']],
              ["Audi", ['A2', 'A1']],
              ["Mercedes", ['E 250', 'S 500']]
            ];
alert(cars.flat());

flat()won't work on strings

When to Use flat():

When working with multi-dimensional arrays and you need to work with them as a flat structure.
When you have an array of arrays and you want to reduce it to a single-level array.
flat()also removes empty slots in arrays (known as sparse arrays)

Array flatMap()

flatMap method in JavaScript is a combination of map() and flat(). It first maps each element of an array using a mapping function and flatten the result into a new array. It flattens the array by one level by default.

const numbers = [1, 2, 3, 4];
alert(numbers.flatMap(num => [num * 2]))

Splicing and Slicing Arrays

The splice() method adds new items to an array.

The slice() method slices out a piece of an array.

Array splice()

const fruits = ['Banana', 'Orange', 'Apple', 'Mango'];
fruits.splice(2, 2, 'Lemon', 'Kiwi');
alert(fruits);

The first parameter (2) defines the position where new elements should be added (spliced in).
The first parameter (2) defines the position where new elements should be added (spliced in).
The rest of the parameters ("Lemon" , "Kiwi") define the new elements to be added.
The splice() method returns an array with the deleted items

Array toSpliced()

ES2023 added the Array toSpliced() method as a safe way to splice an array without altering the original array.
The difference between the new toSpliced() method and the old splice() method is that the new method creates a new array, keeping the original array unchanged, while the old method altered the original array. 

const months = ['JAN', 'FEB', 'MAR', 'MAY'];
alert(months.toSpliced(0,1));
alert(months);
